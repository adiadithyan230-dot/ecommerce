Column Scheme Requirements

For the analytics dashboard to function properly, your CSV dataset must contain columns mapping to these variables:

  • Order ID (identifier)
  • Customer Name (text)
  • Product (text)
  • Category (text classification)
  • Region (geographical node)
  • Quantity (numeric order quantity)
  • Revenue (transaction revenue amount)
  • Payment Method (payment node)
  • Order Status (delivered, cancelled, returned, pending)
  • Order Date (date string)
